📖 Excel 檢視器 - 使用說明

本工具是一個強大的前端 Excel 檔案檢視器，專為快速整合、篩選和匯出多個 Excel 檔案或工作表而設計。所有操作均在您的瀏覽器本機完成，資料不會上傳到任何伺服器，確保資料安全。

主要功能

  • 支援拖放或點擊上傳 .xls 和 .xlsx 檔案。
  • 可一次載入多個檔案，並自訂要匯入的工作表。
  • 提供「列表」和「網格」兩種檢視模式。
  • 強大的資料列操作：即時搜尋、關鍵字勾選、刪除、反選等。
  • 支援多種匯出格式：HTML、XLSX（可選擇全部、僅選取，或合併為單一檔案）。

操作步驟

1. 選擇匯入方式

在「步驟一」中，您可以決定如何從 Excel 檔案（特別是擁有多個工作表的檔案）中讀取資料：

  • 匯入所有工作表：讀取每個檔案中的所有工作表。
  • 僅匯入第一張工作表：預設選項，只讀取每個檔案的第一個工作表。
  • 匯入指定位置：讀取特定位置的工作表。例如輸入 1（第一張）、1, 3（第一和第三張）、2-4（第二到第四張）。
  • 匯入指定名稱：讀取名稱中包含您輸入關鍵字的工作表。
  • 逐一檔案手動決定：上傳檔案後，工具會彈出視窗，讓您手動勾選每個檔案要匯入的工作表。

2. 上傳檔案

在「步驟二」中，您可以透過以下任一方式上傳檔案：

  • 將一個或多個 Excel 檔案拖放到虛線框區域。
  • 點擊「選擇 Excel 檔案」按鈕來選取檔案。

載入後，您可以隨時點擊「🗑️ 清除所有檔案」來重置工具。

3. 檢視與操作

檔案載入後，會顯示完整的「控制面板」和資料表格。

檢視選項

  • 列表檢視：預設模式，所有表格垂直排列，適合詳細檢視。
  • 網格檢視：以卡片形式顯示表格，可使用滑桿調整每行顯示的欄數。在網格模式下，點擊卡片可進入「專注模式」全螢幕檢視該表格。

表格層級操作

您可以對「整個表格」進行批次操作：

  • 勾選所有表格 / 取消勾選：快速選取或取消所有目前顯示的表格。
  • 刪除選定表格：永久移除您所勾選的表格（非表格內的資料列）。

資料列操作（全局）

這些操作會影響「所有表格中」的資料列：

  • 即時搜尋篩選：在輸入框中輸入關鍵字，所有表格會即時隱藏不包含該關鍵字的資料列。
  • 依關鍵字勾選：
    • 輸入單一或多個關鍵字（用空格分隔），點擊按鈕可勾選「同時包含」所有關鍵字的資料列。
    • 若關鍵字用逗號 , 分隔，則會勾選「包含任一」關鍵字的資料列。
    • 可勾選「使用正規表示式」進行更複雜的匹配。
  • 勾選空白列：勾選所有欄位皆為空白的資料列。
  • 勾選全部 / 反向選取：快速全選或反轉目前的勾選狀態。
  • 刪除勾選列：刪除所有表格中目前被勾選的資料列。

全局與匯出

  • 顯示隱藏行列：如果原 Excel 檔中有隱藏的行列，此按鈕會將它們全部顯示出來。
  • 匯出 HTML (全部 / 選取)：將目前顯示的表格（或僅勾選的資料列）匯出為一個 HTML 檔案。
  • 匯出 XLSX (全部 / 選取)：將目前顯示的表格（或僅勾選的資料列）匯出為一個 XLSX 檔案。每個表格會存為一個獨立的工作表。
  • 合併所有表格：將所有表格的資料（包含表頭）合併成「單一一個工作表」並匯出為 XLSX 檔案。
  • 還原始初狀態：清除所有篩選和刪除操作，將資料還原到剛載入時的狀態。

4. 專注模式（網格檢視）

在「網格檢視」下，點擊任一表格卡片，該表格會放大至全螢幕。在此模式下，您可以：

  • 點擊右上角的 × 或按 Esc 鍵退出。
  • 點擊「刪除選取列」僅刪除此表格中的勾選列。
  • 點擊「刪除此表」直接刪除這個表格。
