Excel 檢視與合併工具 - 使用說明

🚀 1. 開始使用：載入檔案

此工具讓您能快速讀取並預覽 Excel (.xls, .xlsx) 檔案內容。

  • 拖放檔案： 直接將一個或多個 Excel 檔案拖曳到上方的虛線框區域。
  • 點擊選取： 點擊虛線框區域，會開啟檔案選取視窗。
  • 匯入模式： 在上傳檔案前，您可以選擇要匯入工作表的模式：
    • 匯入所有工作表： 預設選項，會讀取每個檔案中的所有工作表。
    • 僅第一個工作表： 只讀取每個檔案的第一個工作表。
    • 依工作表名稱： 僅讀取名稱中包含您輸入關鍵字的工作表。
    • 依工作表位置： 僅讀取您指定位置的工作表 (例如：1, 3, 5-8)。
    • 手動選擇： 針對每個檔案，彈出視窗讓您手動勾選要匯入的工作表。
  • 清除檔案： 點擊 清除檔案 按鈕可移除所有已載入的表格，重置介面。

🖥️ 2. 主介面功能

檔案載入後，每個工作表會以一張「卡片」的形式顯示在下方。您可以進行以下操作：

檢視與篩選

  • 切換視圖：
    • 列表視圖 (List)：預設模式，表格垂直排列，適合詳細閱讀。
    • 網格視圖 (Grid)：表格以網格排列。您可以使用右側的 (縮放) 滑桿調整每行顯示的表格數量。
  • 全域搜尋： 在上方的 搜尋... 輸入框中輸入關鍵字，會即時篩選所有表格中包含這些關鍵字的資料列。
  • 重設檢視： 點擊 重設檢視 會清除所有篩選、還原所有被刪除的資料列 (僅限當次載入)，並重置搜尋框。

選取資料列

您可以對表格中的資料列 (row) 進行勾選，以利後續操作。

  • 全選/全不選： 點擊表格標頭 (Header) 左上角的核取方塊，可全選或全不選該表格的可見資料列。
  • 全選所有列：點擊上方控制面板的此按鈕，會勾選所有表格中的所有可見資料列。
  • 反向選取：反轉目前所有表格的勾選狀態。
  • 依關鍵字選取：輸入關鍵字，勾選所有包含該關鍵字的資料列 (支援 Regex、, (或) 或 空白 (且))。
  • 選取空白列：勾選所有內容完全空白的資料列。

刪除資料

  • 刪除表格：
    • (卡片內) 刪除此表：永久刪除這張工作表卡片。
    • (勾選表格) 刪除選定表格：勾選卡片標題旁的核取方塊，再點擊此按鈕，可一次刪除多個表格。
  • 刪除資料列：
    • (卡片內) 刪除選取列：刪除這張卡片中已勾選的資料列。
    • (全域) 刪除選取列：刪除所有表格中已勾選的資料列。

📊 3. 合併檢視與編輯 (核心功能)

當載入多個表格時，合併檢視與編輯 按鈕會啟用。此功能會將所有目前可見的表格合併成一個單一、可編輯的大表格。

注意： 合併時，程式會自動比對所有表格的「標頭 (Header)」。

  • 標頭名稱相同的欄位會被合併到同一欄。
  • 如果表格沒有標頭，程式會自動產生 (欄位 1), (欄位 2) ... 這樣的預設標頭。

資料編輯

  • 編輯資料：點擊後，表格進入編輯模式，您可以直接點擊儲存格來修改內容。
  • 儲存變更 / 取消編輯：儲存或放棄您在編輯模式下所做的修改。
  • 新增資料列：在表格最上方新增一筆空白資料列，並自動進入編輯模式。
  • 複製選取列：複製所有已勾選的資料列，並將其新增到表格頂端。
  • 刪除勾選列：刪除合併表格中所有已勾選的資料列。

欄位操作

  • 排序 (Sort)： 點擊任一欄位的標頭 (例如 "金額") 即可進行升冪/降冪排序。
  • 刪除欄位 (Delete)： 點擊欄位標頭上出現的 × 按鈕，可永久刪除該欄位。
  • 欄位操作：開啟彈窗，您可以在此
    • 全選 / 全不選 欄位。
    • 手動勾選要顯示或隱藏的欄位。
    • 點擊 套用 來更新表格。

資料分析

  • 顯示/隱藏合計：在表格最下方顯示一個「合計列 (Total)」，自動加總所有可辨識為數字的欄位。
  • 搜尋： 主介面的 搜尋... 功能在合併檢視中同樣有效，可快速篩選合併後的大量資料。

📤 4. 匯出功能

您可以將處理後的資料匯出為不同格式。

情境感知： 匯出功能會根據您目前所在的視圖（主介面或合併檢視）而有不同行為。

  • 在主介面：匯出的是多個表格。
  • 在合併檢視：匯出的是單一的合併後表格。

匯出按鈕

  • 匯出 HTML (全部)：將所有可見資料（或合併後的表格）匯出為一個 HTML 檔案。
  • 匯出 HTML (選取)：僅匯出所有已勾選的資料列。
  • 匯出 XLSX (全部)：將所有可見資料（或合併後的表格）匯出為一個 Excel (.xlsx) 檔案。
    • 在主介面時，每個表格會存為一個獨立的工作表 (Sheet)。
  • 匯出 XLSX (選取)：僅匯出所有已勾選的資料列。
  • (主介面) 匯出合併 XLSX：這是一個快捷按鈕。它會**一步到位**，將所有表格合併（包含來源檔案欄位）並直接匯出為一個 Excel 檔案，而不需要先進入「合併檢視」。

⌨️ 5. 快捷鍵

  • Esc 鍵：
    • 如果在「欄位操作」彈窗開啟時按下，會關閉彈窗。
    • 如果在「合併檢視」中按下，會關閉合併檢視。
    • 如果在「網格視圖」的放大預覽中按下，會關閉預覽。
