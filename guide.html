Excel 檔案彙整與進階處理工具 - 使用說明

本工具專為處理大量、格式複雜（如政府機關/財務報表）的 Excel 檔案而設計，提供從載入、清洗、去重到合併匯出的一站式解決方案。

⚙️ 1. 步驟一：選擇匯入方式與清洗設定

在載入檔案前，您可以預先設定強大的「自動清洗」規則，系統會在讀取瞬間幫您整理好資料。

  • 選擇匯入模式：
    • 僅匯入第一張工作表： (預設) 只讀取每個檔案的第一個工作表（最常用）。
    • 匯入指定位置： 僅讀取您指定位置的工作表 (例如輸入：1, 3, 5-8)。
    • 匯入指定名稱： 僅讀取名稱中包含您輸入關鍵字的工作表。
    • 匯入所有工作表： 讀取檔案中的每一張工作表。
    • 逐一檔案手動決定： 載入時彈出視窗，讓您針對每個檔案手動勾選。
  • 預處理清洗設定 (解決複雜表頭與垃圾資料)：
    • ✂️ 拋棄與壓縮表頭： 您可以設定「先拋棄前 X 列」（去除表名、單位等無用資訊），並「將接下來的 Y 列壓縮成完美表頭」。系統會自動將多列標題合併並換行顯示，遇到同名欄位還會自動編號（如 預算數_2），徹底解決資料被覆蓋或遺失的問題。
    • 🗑️ 移除空白列： 自動過濾掉整列沒有任何內容的無效資料。
    • 🚫 剔除關鍵字： 匯入時自動刪除包含特定字眼（如：「小計」、「總計」）的列。
  • 載入檔案： 設定完成後，將 Excel (.xls, .xlsx) 檔案拖放到虛線框，或點擊按鈕選取檔案，系統即會以極速模式完成載入。

🖥️ 2. 主介面功能 (雙軌模式)

檔案載入後，每個工作表會以「卡片」形式呈現。現在您不需要進入合併檢視，就能在主畫面進行強大的批次預處理。

👁️ 檢視與表格操作

  • 切換視圖： 支援「列表 (List)」與「網格 (Grid)」視圖，網格視圖可透過滑桿調整顯示欄數。
  • 👁️ 欄位操作： 點擊此按鈕，可預先「隱藏」不需要比對的欄位，讓主畫面更清爽（此隱藏設定會延續至合併檢視中）。
  • 依基金名稱排序： 根據系統預設的清單，自動將所有表格卡片進行排序。
  • 刪除選定表格： 勾選多張卡片右上角的核取方塊，可一鍵批量刪除整個工作表。

▶️ 基礎資料列操作 (篩選與勾選)

您可以透過左側的 Checkbox 勾選特定資料列，以進行後續的複製、刪除或合併。全選框已加入置頂與靠左釘選設計，捲動時不會消失。

  • 即時搜尋篩選： 輸入關鍵字，畫面會瞬間隱藏不包含該字眼的資料列。
  • 依關鍵字勾選： 輸入關鍵字並點擊勾選，支援多組關鍵字（空格代表 AND，逗號代表 OR）及 Regex 正規表示式。
  • 反向選取 / 勾選空白列 / 勾選全部： 快速輔助您選取目標資料。
  • 刪除勾選列： 將主畫面中所有被打勾的資料列一併刪除。

🚀 進階資料列操作 (主畫面直接處理)

  • ✨ 智慧去重： 點擊後選擇一個識別欄位（如「基金名稱」）。系統會自動比對該名稱與「來源檔案名稱」，若發現資料來源與名稱不符的舊資料/殘留資料，會自動幫您打勾標記，方便您一鍵刪除。
  • ✍️ 編輯資料 / 💾 儲存： 點擊後，主畫面所有表格皆可直接點擊修改內容，修改完畢記得點擊儲存。
  • 📋 複製列： 將打勾的資料列原地複製一份。

🧮 依欄位條件篩選勾選 (複合篩選)

這是一套強大的邏輯篩選器：

  1. 選擇 條件一 的欄位（例如："本年度預算數"），並設定規則（如："為零"、"空值"、"有值"、"包含" 等）。
  2. 若需更精準的比對，可點擊右上角 「+ 新增第二條件」 展開條件二，並選擇邏輯交集（AND 且 / OR 或）。
  3. 點擊 「🚀 執行進階篩選勾選」，系統會自動幫所有符合嚴格條件的資料打勾。

📊 3. 合併檢視與編輯 (核心處理)

處理完畢後，您可以將所有分散的表格融合成一張大總表。

  • 進入方式：
    • 合併檢視： 將主畫面所有可見資料合併。
    • 勾選資料列合併檢視： 僅合併主畫面中有被打勾的資料列。

(註：如果系統偵測到您上次有未完成的合併編輯進度，會貼心詢問是否要接續上次狀態。)

頂部工具列操作：

  • 新增來源欄位： 在表格最左側動態加入/移除「來源檔案」欄位。
  • 合計列： 在表格最底部產生一列，自動加總所有純數字欄位。
  • 新增列 / 複製列 / 刪除列： 在合併總表中進行資料列的增刪。
  • 智慧去重 / 欄位操作 / 編輯資料： 與主畫面相同，您也可以在合併後再次執行這些操作。
  • 快速排序與刪除欄位： 直接點擊表格上的標題名稱可進行「升冪/降冪」排序；點擊標題旁邊的 × 可直接刪除該欄位。

(合併檢視內同樣具備可收合的「🧮 複合條件篩選器」與「即時搜尋列」，方便您在總表中二次清洗資料。)

📤 4. 匯出功能

  • (主畫面) 合併所有表格： 快捷操作！不需要進入合併檢視，直接將畫面上所有看得到的表格，加上「Source File」來源標記後，一鍵打包匯出成一個完整的 .xlsx 檔案。
  • (合併檢視) 匯出檢視： 當您在合併檢視中完成了所有的編輯、隱藏欄位、排序與加總後，點擊此按鈕，系統會將您「肉眼所見」的最終成果完美匯出成 Excel 檔。

⌨️ 5. 快捷鍵與系統守護

  • 🔄 復原 (Undo)： 按下 Ctrl + Z (Mac 為 Cmd + Z) ，系統具備強大的歷史記憶功能。無論您是刪除了資料列、刪除了欄位、執行了智慧去重標記，或是儲存了錯誤的編輯，都能立刻倒退回上一步！
  • Enter 鍵快速執行： 在輸入關鍵字或篩選條件時，按下 Enter 鍵即可直接執行篩選或勾選，不需移動滑鼠點擊按鈕。
  • Esc 鍵快捷關閉： 可用來快速關閉「欄位操作彈窗」、「智慧去重設定」、「合併檢視總表」或「網格視圖的放大預覽」。
