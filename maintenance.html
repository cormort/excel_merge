🛠️ 程式碼維護與註解

此頁面說明本工具的程式碼結構，方便您未來進行檢視、修改或維護。

程式碼結構

本工具採用「外層框架 + 內嵌應用」的結構，將頁籤導覽與核心功能分離。

1. 外層頁籤系統 (根目錄)

這部分負責管理三個主要頁籤的切換和內容載入。

  • index.html: 整個專案的主入口點。它包含了三個頁籤的 HTML「外殼」，以及載入 Excel 檢視器的 <iframe>。
  • style.css: 負責 index.html 中頁籤導覽列 (tab-nav) 和容器 (tab-container) 的外觀樣式。
  • script.js: 負責頁籤的點擊切換邏輯。它會動態抓取 guide.html 和 maintenance.html 的內容，並將其載入到對應的頁籤面板中。
  • guide.html / maintenance.html: 靜態的 HTML 內容頁面，本身不含 script，僅作為內容被 script.js 讀取。

2. 核心應用程式 (app/ 資料夾)

這是「🚀 Excel 檢視器」頁籤中，透過 <iframe> 載入的獨立應用程式。

  • app/index.html: 核心應用的 HTML 結構。定義了上傳區域、控制面板、表格顯示區域，以及合併檢視的彈窗 (modal)。
  • app/style.css: 核心應用的所有 CSS 樣式。負責美化表格、按鈕、彈窗等所有介面元素。
  • app/script.js: (主要邏輯) 所有的功能程式碼都在這裡。使用 IIFE (Immediately Invoked Function Expression) 模式封裝在 ExcelViewer 物件中，包含了檔案讀取 (XLSX.js)、表格渲染、合併檢視、資料編輯、篩選、匯出等所有核心邏輯。

維護建議

  • 修改 App 功能： 如果您要修改「Excel 檢視器」的功能 (例如新增按鈕、修改合併邏輯)，您幾乎*只需要*修改 app/ 資料夾中的檔案 (app/index.html, app/style.css, app/script.js)。
  • 修改說明文字： 如果您要修改「使用說明」或「程式碼維護」的文字，請分別編輯 guide.html 或 maintenance.html 檔案即可。
